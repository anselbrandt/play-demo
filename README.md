# Play Web Framework Demo

### Scafolded with:

```
sbt new playframework/play-scala-seed.g8
```

### Run

```
sbt run
```

### Static Assets

Assets in the `./public` folder will be located at:

`http://localhost:9000/assets/`

ex.

`http://localhost:9000/assets/mountbaker.jpg`
